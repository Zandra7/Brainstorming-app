# Brainstorming-app

## Hvordan kjøre appen
1. Klon prosjektet fra github
2. Skriv "npm install" for å innstallere avhengigheter - Nå skal du ha fått en mappe kalt "node_modules"
3. Skriv "node index.js" i terminalen for å kjøre appen (funker ikke)
